# React useEffect – 10 Ejercicios Prácticos

Este repositorio contiene una serie de **10 ejercicios progresivos** para aprender el hook `useEffect` en React, 
desde lo más básico hasta casos de uso más reales.

## 🎯 Objetivo

Comprender cómo funciona `useEffect` en escenarios reales:

- Ejecución de efectos
- Manejo de dependencias
- Cleanup (limpieza de efectos)
- Timers (`setTimeout`, `setInterval`)
- Simulación de consumo de APIs
- Sincronización con el DOM (`document.title`)
- Optimización (debounce)

---

## 🧠 ¿Qué aprendí?

- Cuándo y por qué usar `useEffect`
- Cómo evitar ejecuciones innecesarias
- Cómo limpiar efectos correctamente
- Separación de responsabilidades en múltiples `useEffect`
- Problemas comunes como:
  - intervalos duplicados
  - closures (valores antiguos)
  - mala gestión de dependencias

---

## 🧪 Ejercicios incluidos

1. Actualizar `document.title` con estado
2. Ejecutar efecto solo al montar
3. Detectar cambios en input
4. Contador automático con `setInterval`
5. Cleanup de intervalos
6. Montaje y desmontaje de componentes
7. Simulación de fetch (API)
8. Dependencias dinámicas (búsqueda)
9. Debounce con `setTimeout`
10. Múltiples efectos (timer, title, localStorage)
